# FPSDeathGame
1인칭 서바이벌 데스게임
</br>
적AI를 죽일 때 마다 1점을 얻으며, 총 NN점 획득 시 승리하며 게임은 즉시 종료됨.
</br>
게임 종료 전까지 적AI는 물론 플레이어 역시 무한 리스폰됨
